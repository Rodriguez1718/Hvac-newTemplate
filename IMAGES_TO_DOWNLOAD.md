## 🚀 Quick Start Workflow

### 1. Download Images
```bash
# Create temp folder
mkdir temp-downloads
cd temp-downloads

# Download images using browser "Save Image As"
# Or use wget/curl if you prefer command line
```

### 2. Optimize All at Once
```bash
# Start dev server
npm run dev

# Open optimizer
# http://localhost:4321/image-optimizer.html

# Batch process:
# - Upload each image
# - Set appropriate size and quality
# - Download optimized version
# - Save to public/images/ with correct name
```

### 3. Recommended Settings

| Image Type | Width | Quality | Example |
|------------|-------|---------|---------|
| Hero backgrounds | 1920px | 75% | hero-slide-1.webp |
| Content images | 1200px | 80% | quality-hvac-service.webp |
| Portfolio/Gallery | 800px | 80% | portfolio-1.webp |
| Service cards | 600px | 80% | iaq-service-1.webp |
| Promo tiles | 800px | 80% | promo-maintenance.webp |

### 4. Update JSON Files

After saving all images to `public/images/`, update your JSON files:

**Example for index.json:**
```json
{
  "images": [
    "/images/hero-slide-1.webp",
    "/images/hero-slide-2.webp",
    "/images/hero-slide-3.webp"
  ]
}
```

**Example for portfolio:**
```json
{
  "images": [
    "/images/portfolio-1.webp",
    "/images/portfolio-2.webp",
    "/images/portfolio-3.webp",
```
